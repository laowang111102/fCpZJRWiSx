### 系统角色

- 管理员
- 用户

- 首页概览

  - 新增
  - 编辑
  - 修改
  - 借阅
- 图书类型管理
  - 搜索
  - 添加
  - 编辑
  - 删除
- 借阅信息管理
  - 新增
  - 修改
  - 归还
- 用户管理
  - 搜索
  - 新增
  - 修改
  - 查询
- 密码更改

#### 用户功能

- 注册登录
- 首页展示
- 图书管理
  - 搜索
  - 借阅
- 借阅图书管理
  - 归还
- 密码修改

## 运行环境

- 后端环境：Java 1.8+，MySQL 5.7+
- 前端环境：Node.js 12.0+

## 部署步骤

1. 准备开发环境
2. 导入数据库SQL文件
3. 部署后端服务
4. 部署前端服务

## 目录结构

```
- backend                // 后端代码目录
  - src
    - main
      - java
      - resources
- frontend               // 前端代码目录
  - src
    - assets
    - components
    - views
- database               // 数据库文件目录
  - schema.sql           // 数据库结构文件
  - data.sql             // 数据库初始化数据文件
- docs                   // 参考文档目录
  - README.md
```
